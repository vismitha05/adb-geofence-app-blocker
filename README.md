Geofencing for Restricted Areas

A Python-based system that restricts mobile device functionality when a device enters a sensitive or restricted area using Android Debug Bridge (ADB).

#Overview
This project implements a geofencing mechanism to control mobile phone usage in restricted zones. When a device enters the defined area, specific functionalities are limited through ADB commands to ensure security and compliance.

#Features
Automated mobile restriction in restricted areas
Device control using Android Debug Bridge (ADB)
Python-based implementation for easy automation
Suitable for sensitive and secure environments

#Tech Stack
Python
Android Debug Bridge (ADB)

#Use Case
Restricted zones
Secure environments

Controlled mobile usage areas
